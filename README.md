#  CHEZ XAVIERA - Page Produit E-commerce

## Description

Ce projet est une **page produit e-commerce** développée en HTML, CSS et JavaScript.
Il met en avant un article de la collection *granny* avec une interface élégante, inspirée d’un design moderne et épuré.

---

## Fonctionnalités

###  Affichage produit

* Image principale du produit
* Miniatures cliquables permettant de changer l’image principale

###  Informations produit

* Nom du produit
* Collection
* Prix unitaire et prix réduit (achat en lot)
* Description détaillée
* Avis clients avec étoiles 

### Gestion de quantité

* Boutons **+ / -** pour modifier la quantité
* Valeur minimale contrôlée

###  Interaction utilisateur

* Bouton **Ajouter au panier**
* Message de confirmation dynamique
* Animation du bouton lors du clic

###  Informations supplémentaires

* Livraison au Cameroun
* Modalités de paiement
* Origine du produit

---

##  Technologies utilisées

* **HTML5** → Structure de la page
* **CSS3** → Mise en page et design
* **JavaScript (Vanilla)** → Interactions dynamiques

##  Installation et utilisation

1. Télécharger ou cloner le projet
2. Placer les images dans le même dossier que le fichier HTML
3. Ouvrir `index.html` dans un navigateur

---

##  Améliorations possibles

* Ajouter un **vrai système de panier (localStorage)**
* Créer une **page panier complète**
* Ajouter un **backend (PHP, Node.js, etc.)**
* Rendre la page **responsive (mobile/tablette)**
